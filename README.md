🎯 Counter App
Overview

A simple interactive counter application built with JavaScript to practice DOM manipulation, event listeners, conditions, and state updates.

✨ Features
Increment counter
Decrement counter
Reset counter
Dynamic color updates:
Green for positive numbers
Red for negative numbers
Black for zero
🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6)
📚 Concepts Practiced
Variables and state
Functions
Conditions
Event listeners
DOM manipulation
Dynamic UI updates
🚀 Goal

This project was created to strengthen my JavaScript fundamentals and improve my understanding of interactive web applications.
